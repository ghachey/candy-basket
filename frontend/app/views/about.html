About Candy Basket

The candy basket software is modelled on 'del.icio.us' but with a focus private use within an organization. Candy Basket was developed as a result of PiPP's work on changing the political environment in the Pacific. This work changed the way that PiPP looked at monitoring information, taking it away from more traditional methods and focussing more on Outcome Mapping.

The source code of the Candy Basket project will be freely available. It is extensively documented making it as easy as possible for anyone interested in using, deploying and developing Candy Basket to do so.

  • The user documentation contains information for users and describes how to use the tool.
  • The administrator's documentation is for anyone with the necessary technical skills to deploy the application within their organization
  • The developer's documentation should be read in full by anyone wanting to contribute to the development of Candy Basket.
